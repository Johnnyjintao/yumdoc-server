## 一、项目介绍

- 技术栈：Nodejs, MySQL; Koa2, Sequelize, jwt.
- 用nodejs的koa2框架搭建接口服务器，提供API接口；
- 接口包含：
  - jwt做权限接口验证。
  - 用户登录注册接口；
  - 文章增删改查，搜索，分页接口；
  - 分类增删改查接口；
  - 分类关联多文章。
- Sequelize管理mysql数据库。

### 项目使用
- 1. 在根目录下进入项目，
- 2. 安装包，执行: `npm install` 命令，
- 3. 启动服务: `npm start`; 监听地址：http://localhost:3000/ 即可以访问。
